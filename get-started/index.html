      # 快速上手

      安装完成之后，在vue文件中引入需要的组件进行使用即可。

      # 示例

      import Vue from 'vue'
import {Button} from 'yz-ui'

new Vue({
  el: '#app',
  components: {
    'y-button': Button
  }
  render: h => h(App)
})

      # 提醒

      • 样式文件需要单独引入

      • 使用该组件库时，需要使用 border-box 盒模型，否则会影响样式；

      * {
  box-sizing: border-box;
  padding: 0;
  margin: 0;
}
